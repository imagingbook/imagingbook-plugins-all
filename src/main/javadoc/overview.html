This is a collection of ImageJ plugins referenced in the authors' textbooks on digital image processing published by Springer-Verlag in various languages and editions (see www.imagingbook.com for additional details). Plugin classes are grouped by topic inside dedicated Java packages, each package typically associated with a particular book chapter.

Dependencies:

Depends on ImageJ and the ImagingBook Common library: imagingbook-common.jar should be placed in ImageJ/jars/.

Installation:

Place the supplied JAR file in the ImageJ/plugins/ folder and restart ImageJ. All available plugins should show up in ImageJ's Plugins menu.
